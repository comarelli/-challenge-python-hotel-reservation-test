No desafio foi feita a implementação de um código para a escolha de um hotel para reserva.

O código foi feito de forma genérica, tornando adapatações futuras mais fáceis de serem implementadas.

A primeira parte do código é a lógica para contabilizar quantos dias de semana e de fim de semana a pessoa irá ficar.

Em seguida é feito o cálculo do preço que irá ficar cada hotel.

Por último, e a maior parte do código, é feita a lógica para definir qual será a melhor opção de hotel.

Os resultados passaram nos testes fornecidos.
